# Vo Thi Thuy Linh Portfolio

One-page bilingual React portfolio for an F&B service professional, built with Vite and plain CSS. The site includes English/Vietnamese content and a theme toggle for light and dark modes.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The site is ready to deploy on Vercel. Public SEO assets are in `public/`, including `robots.txt`, `sitemap.xml`, and `og-image.png`.
